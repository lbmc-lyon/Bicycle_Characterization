The following packages are needed :

conda :
 - python = 3.10
 - notebook
 - pip 
 - git
 - scipy

pip :
 - pip install opencv-python
 - pip install git+https://github.com/sushil-bharati/pyzbar.git

Once you installed all the packages, you have to copy the 2 .dll files in this path : 
  C:\Users\username\AppData\Local\anaconda3\envs\your_env_name\Lib\site-packages\pyzbar\

The 2 files are : 
 - libiconv.dll
 - libzbar-64.dll
